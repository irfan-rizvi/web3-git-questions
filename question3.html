Question 3: Review 3 different GUI Git clients

There are many GUI interfaces available in the market for Git. Some are free to use and some are paid versions. Following is a list of Git GUI interfaces.

SmartGit
It can be used for free for teacher and students. It has a very clean and straight-farward user interface and one of the best rated GUI interface for Git.
GitKraken
It has very sleek and beautiful user interface. It has subscription based pricing which might not be very attractive to most people.
SourceTree
This GUI interface for Git is free to use. The user interface is not that great but it can perform advanced Git operations. SourceTree divides changes to be committed in chunks and allows each chunk to be committed separately.
